How to Set up & Use Atom as a MarkDown Editor in Windows 10

What is Atom?

Atom is a desktop application built with HTML, JavaScript, CSS, and Node.js integration. It runs on Electron, a framework for building cross platform apps using web technologies.

Why using Atom?

Atom has everything you would expect:

  1. Cross-platform editing:
    Atom works across operating systems. Use it on OS X, Windows, or Linux.

  2. Built-in package manager:
    Search for and install new packages or create your own right from Atom.

  3. Smart autocompletion:
    Atom helps you write code faster with a smart and flexible autocomplete.

  4. File system browser:
    Easily browse and open a single file, a whole project, or multiple projects in one window.

  5. Multiple panes:
    Split your Atom interface into multiple panes to compare and edit code across files.

  6. Find and replace:
    Find, preview, and replace text as you type in a file or across all your projects.

Installation of Atom

  1. Visit Atom.io
  2. Click Download
  3. Install it

If this is the first time to install and to open Atom, it will jump many panes including welcome page and etc. Just close them.

How to Work under a Project?

In a multiple-document project, it is a much better feeling if all relevant files are in one place. Thus, all images, contents, and scripts can be selected fluently. Atom allows this.

Atom Project Folders

To view a whole folder:

  1. Click File>>Open
  2. Navigate to the folder you want to go
  3. Donâ€™t click the file, but only to open the folder
    Atom then opens the desired folder, listing all files contained in it.

Setting up MarkDown for Atom in Windows 10

  1. Open Atom
  2. Click Packages>>Setting View>>Manage Packages
  3. On the left of Settings pane, click install
  4. Now, it is ready to install packages need for MarkDown.

Atom Package Manager

Installation of MarkDown Packages

There are three packages need for Atom MarkDown:

  1. language-markdown
  2. markdown-writer
  3. markdown-preview

Then, you can edit and preview your document without leaving Atom.

How to open MarkDown Preview?

  1. Open MarkDown file needs to be edited.
  2. Click Packages>>Markdown Preview>>Toggle View
  3. Then, the preview pane will be shown on the right.
